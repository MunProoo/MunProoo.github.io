## 🖐️ 웹 3.0이란
**웹 3.0**은 **인공지능**과 **블록체인**을 기반으로 맞춤형 정보를 제공하고 **데이터 소유를 개인화**하는 **3세대** 인터넷이다.   
기존에 사용하던 서비스에서 인공지능을 통해 개인 맞춤형 정보를 얻고, 블록체인을 통해 개인의 정보 소유 및 보안을 강화하는 **지능형 웹 기술**이라 할 수 있다.    

인공지능과 블록체인 두 가지의 제일 핫한 분야가 접목된 웹 기술이라니 벌써 설렌다.   

## 🖐️ 웹 1.0과 웹 2.0
웹 기술을 시대적으로 구분하면 과거에는 웹 1.0이 있었다. 현재는 웹 2.0의 시대이며, 앞으로 다가올 미래의 웹 기술은 웹 3.0이다.    
각 세대의 특징과 차이점을 알아보자.   

### 🖐️ 웹 1.0 (1991 ~ 2004)
**웹 1.0**은 `읽기`중심의 초기 인터넷 기술이었다. 과거의 웹 페이지는 해당 페이지의 소유자가 제공하는 정보만을 확인할 수 있었다.   

![](/assets/img/YY-MM/2024-02-21-20-30-55.png)   

<br>

**웹 1.0 특징**
- 정적 웹페이지   
- 사용자 생성 콘텐츠 전무

### 🖐️ 웹 2.0 (2004 ~ 현재)
**웹 2.0**은 이전의 `읽기`중심의 인터넷 기술에서 `쓰기`가 가능해졌다. 그에 따라 다양한 정보의 상호작용이 가능해졌다.   

웹상에서 다양한 사람과 양방향 커뮤니케이션이 가능한 웹 2.0은 스마트폰의 보급으로 인해 폭발적으로 활성화되었다.    
JavaScript, HTML5, CSS3을 지원하며 이를 통해 유튜브, SNS, 이커머스 등 **플랫폼 기반 웹 2.0**이 완성되었다.   

![](/assets/img/YY-MM/2024-02-21-20-41-21.png)   

**웹 2.0**은 기업에서 데이터를 소유하고 배포하는 **중앙화된 인터넷 세대**이다. 사용자는 브라우징을 통해 정보를 얻을 수 있고, 기업은 서비스를 제공하는 대신 사용자의 정보를 받는다. 기업들은 이런 방식으로 얻은 데이터를 타겟 광고에 사용한다.   

자유로운 커뮤니케이션이 가능한 웹 2.0에선 정보 공유를 비약적으로 높여줬지만, **개방성이 낮아졌다.**   

웹 2.0에서는 **우리가 만든 정보가 플랫폼에 의해 통제**된다.    
특정 플랫폼에 개인정보가 집중되면서 개인정보의 유출 사건도 왕왕 발생한다.   

## 🖐️ 웹 3.0
**웹 3.0**은 정보의 `읽기와 쓰기`를 넘어 `소유`를 개인이 가능하게 만들어 개인화된 인터넷 환경을 구성하고자 한다. 웹 2.0과 웹 3.0의 가장 큰 차이는 **정보를 누가 소유하느냐**인 것이다. 웹 2.0에서 플랫폼이 소유하고 있던 정보들을 웹 3.0에서는 **개인이 소유**하게 된다.   


웹 3.0에서 블록체인기술을 통해 정보를 보호하고 개인이 정보를 소유하게 됨으로써 웹 2.0과 달리 `보안성`과 `익명성`이라는 장점을 얻을 수 있다. 웹 3.0에서는 P2P(Peer to Peer) 환경이 가능해 정보를 탈취하려면 네트워크 전체를 해킹해야 하므로 사실상 해킹이 불가능하다.   

![](/assets/img/YY-MM/2024-02-21-21-07-13.png)

## 🖐️ 웹 3.0과 블록체인
**웹 3.0** 구성에 블록체인이 주목받는 이유는 정보의 암호화와 분산화를 통해 보안성을 높일 수 있는 장점때문이다. 앞서 언급한 사용자 간의 P2P도 블록체인으로 구성이 가능하며, 블록체인의 특성으로 위변조도 어렵다. 또한 개인이 정보를 소유하게 되기 때문에 **운영 주체가 플랫폼이 아닌 네트워크로 변경**된다.   

현재도 웹 3.0 서비스들이 존재한다.    
웹 브라우저 `Brave`는 브랜드에 제공하는 광고의 양을 개인이 제어한다. 정보 주체가 개인으로 이동함으로써 정보 제공자가 플랫폼이 아닌 개인이 되면서 광고의 노출과 성과로 인한 수익이 개인에게 분배된다.   

웹 3.0에선 플랫폼에 집중되던 정보가 다시금 개인에게 흩어지는 탈중앙화가 이뤄진다.

## 🖐️ 웹 3.0의 한계와 초기 비판
일부 초기 비평가들은 탈중앙화된 웹의 실현 가능성에 대해 회의적이다. 또 다른 이들은 블록체인 기술이 효능을 사용해보고 판단하기에는 너무 초기 단계에 있다고 주장한다.    

하지만 우리가 웹 3.0을 통해 할 수 있는 것들은 웹 2.0에 익숙한 우리가 생각조차 못하고 있을 수 있다.    
웹 3.0에 구출되는 서비스와 앱은 현재의 시각에서는 말이 되지 않을 것이다.    

주인장은 웹 3.0이 당장의 대중화는 힘들 수 있어도 결국 도래할 것이라 믿는다.   

> 1943년 IBM 회장은 다음과 같이 말했다.   
> "I Think there is a world market for maybe five computers. "   
> "컴퓨터는 전세계적으로 5대만 수요가 있을 것이다."
