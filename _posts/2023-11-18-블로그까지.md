#과정
- chirpy 테마 fork
- 로컬에 git clone
- ruby, npm, jekyll install
  - 각각 init, bundle 실행위해서
- `tools/init` 으로 chirpy 테마 적용을 위한 초기 작업 실행
- bundler 작업
- _config.yml 에서 각종 설정작업
- `jekyll serve` 로 로컬 구동
- git push
