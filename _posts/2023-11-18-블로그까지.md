#과정
1. chirpy 테마 fork
2. 로컬에 git clone
3. ruby 설치 
   - `brew install rbenv`
   - `rbenv install 3.0.6`
   - `rbenv rehash` 설치된 버전 실행
   - `rbenv global 3.0.6` 선택한 버전으로 전역지정
4. `tools/init` 으로 chirpy 테마 적용을 위한 초기 작업 실행
5.  bundler 작업
6. _config.yml 에서 각종 설정작업
